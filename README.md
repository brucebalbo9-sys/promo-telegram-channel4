# Презентация Telegram-канала Алевтины

Это обычный статический сайт без сборки, базы данных и регистрации. Он открывается прямо из файла `index.html` и подходит для GitHub Pages.

## Как посмотреть

1. Откройте файл `index.html` двойным кликом.
2. На первом экране нажмите `Посмотреть, что внутри`.
3. Попробуйте три варианта выбора тем.
4. Пройдите мини-тест.
5. Нажмите кнопку подписки на Telegram-канал.

## Где поменять ссылки

Ссылки лежат в файле `config.js`.

В исходном файле `01-links.txt..txt` адрес Telegram был записан с опечаткой `https:https://`. В сайте используется исправленный адрес:

`https://t.me/Alevtina_designer_Pankova`

## Файлы сайта

- `index.html` — страница сайта.
- `styles.css` — внешний вид.
- `script.js` — интерактивный выбор и мини-тест.
- `config.js` — ссылки Telegram и ВК.
- `favicon.svg` — иконка вкладки.
- `og-cover.jpg` — обложка для соцсетей.
- `.nojekyll` — файл для GitHub Pages.
- `assets/site/` — web-копии изображений с латинскими именами.

Исходные фотографии в папке `assets` не удалялись и не переименовывались.
